# AIoT Demo Code

This folder contains a lightweight Python simulation that illustrates the core
ideas from the AIoT case study report.

## What this demo shows
- Synthetic sensor data generation representing the device layer
- Edge preprocessing and anomaly detection
- Cloud-style baseline training and model updates
- Telemetry payload formatting for secure, structured reporting

## Run the demo

```bash
cd code
python3 aiot_demo.py
```

## Notes
- The demo is intentionally dependency-free and runs on standard Python.
- It is designed to reflect the report's architecture and real-world AIoT patterns.
- You can extend it with real sensor input, TinyML model inference, or MQTT-based telemetry.
